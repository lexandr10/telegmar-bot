# Telegram Bot (Cloudflare admin)

## Folders
- \`backend/\` — Express + TypeScript API
- \`admin/\` — React admin panel (MUI)

## Quickstart

### Backend
cd backend
yarn install
yarn dev
# env variables — see backend/.env.example

### Admin
cd admin
yarn
yarn dev
# env variables — see admin/.env.example
